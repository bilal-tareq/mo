# Mo Project

A full-stack application consisting of:
- **Backend:** Django REST Framework API with PostgreSQL/Supabase database.
- **Frontend:** Vue 3 application built with Vite, Pinia, and Vue Router.

## Directory Structure
- `/backend` - Django application
- `/frontend` - Vue 3 client application
